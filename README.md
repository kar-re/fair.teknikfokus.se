# fair.teknikfokus.se
A CMS system for doing digital career fairs

## Installation

Install and setup a PostgreSQL database.

Install npm dependencies.
```bash
$ npm install
```

Copy ```.env.example``` to ```.env```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
